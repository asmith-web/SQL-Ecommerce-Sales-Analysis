# SQL E-Commerce Sales Analysis

## Project Overview

This project analyzes the Brazilian Olist E-Commerce dataset using MySQL.

The objective is to solve real-world business problems using SQL and generate insights related to customers, products, sellers, payments, and orders.

---

## Tools Used

- MySQL
- SQL

---

## Dataset

Brazilian Olist E-Commerce Dataset

Tables used:

- Customers
- Orders
- Order Items
- Products
- Sellers
- Payments

---

## SQL Concepts Used

- SELECT
- WHERE
- ORDER BY
- GROUP BY
- Aggregate Functions
- HAVING
- INNER JOIN
- LEFT JOIN
- CASE WHEN
- Subqueries
- String Functions

---

## Business Questions Solved

- Total customers
- Total orders
- Top-selling product categories
- Revenue by state
- Seller performance
- Customer segmentation
- Average payment analysis
- High-value orders
- Products above average weight

More than **50 SQL business queries** were written during this project.

---

## Key Insights

- Identified top-performing product categories.
- Analyzed customer purchasing behavior.
- Evaluated seller revenue performance.
- Compared revenue across Brazilian states.
- Generated business insights using SQL.
